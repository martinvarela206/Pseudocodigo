# Pseudocodigo a C

App web para escribir pseudocodigo con resaltado de sintaxis y ver a la derecha el flujo de ejecucion y la traduccion a C.

## Funciones

- Editor con resaltado de sintaxis y diagnosticos basicos.
- Diagrama de flujo lineal del programa principal.
- Traduccion a C (puro) de las instrucciones soportadas.

## Uso

- Instalar dependencias: `npm install`
- Ejecutar en desarrollo: `npm run dev`
- Compilar: `npm run build`
